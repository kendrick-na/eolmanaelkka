## 0. 한 줄 정의

**"청첩장·부고를 붙여넣으면 — 얼마 낼지, 봉투에 뭐라 쓸지, 뭐라 인사할지를 한 번에 끝내고, 당신의 경조사 관계까지 기억하는 카톡 비서."**

## 1. 왜 이 서비스인가 (페인 — 실증)

### 진짜 페인 순위 (검증됨, 국룰이 방어선인 '계산'보다 예절·관계가 더 두꺼움)
1. **봉투 문구·조문 예절 모름** (가장 두꺼운 페인) — "축의금/조의금 봉투 쓰는 법" SEO 블로그 수백 건 양산 = 반복 검색 수요. 사회초년생 필수, 종교별 분기(기독/천주교/불교)로 갈수록 어려움.
2. **애매한 관계·비국룰 구간 판정** — "15만·30만은 어떤 사이" 논쟁이 커뮤니티 상시. 호텔 식대 15만이면 10만은 손해 프레임.
3. **종교별 조문 멘트 불안**
4. 순수 액수 계산 (국룰 5만/10만로 상당 커버 = 단독으론 약함)

### 금액적 페인 규모 (실증)
- 축의금 평균 **11.7만원(2025)**, 3년새 7.3만→11.7만 (**+60%**)
- "5만원 국룰 옛말" 반복 보도(서울신문·세계일보·파이낸셜뉴스)
- 직장인 연간 경조사비 ~140만원 (추정)
- 방송 논쟁(엄지인 5만 vs 박명수 20만 vs 전현무 30만)이 전국 화제 → "국룰로 안 끝난다"는 증거

### 기존 해결책의 공백 (= 우리 니즈 근거)
| 지금 사람들이 푸는 법 | 불편함 |
|---|---|
| 네이버 "축의금 얼마" 검색 | 블로그마다 답 다름, **내 식장·내 관계 반영 안 됨** |
| 봉투 쓰는 법 블로그 정독 | 매번 다시 검색, 한자 이미지 따라 씀 |
| 단톡방에 물어봄 | 눈치 보임, 관계 노출, 답 제각각 |
| 카카오페이 축의금 봉투 | **송금만** 됨. 얼마·문구·예절은 본인 몫 |

**결정적 공백**: 모바일 청첩장에 카카오페이 QR이 붙는 게 표준화됨 = **"초대장 → 송금"의 앞단(얼마·문구·멘트)이 통째로 비어 있음.** 카카오페이도 2025 만우절에 "축의금 자동측정"을 개그로만 내놓고 실제 출시 안 함 = 수요는 검증됐으나 방치된 무주공산.

## 2. 무엇이 다른가 (해자 — 냉정 검증 통과분만)

### ⚠️ 얕은 차별점 (솔직히 인정 — 전면에 세우지 않음)
- "실시간 식대" → Mowatool이 이미 지역별 식대 가중. 웨딩홀 식대 실시간 API 부재. **"공개 통계 인용"으로 정직 표현**(실시간 워딩 금지).
- "손없는날" → 음력 끝자리 한 줄 계산(30분). 누구나 따라옴.
- "원스톱 통합"·"MCP 채널"·"초대장 파싱" → 선점값이나 배타성 없음(2주면 따라옴).

### ✅ 진짜 해자 = 관계 히스토리 원장 (복제 불가)
**"내가 이 사람에게 얼마 냈나 / 이 사람이 나에게 얼마 냈나"가 쌓이면**, 다음 계산이 "일반 평균"이 아니라 **"우리 관계의 호혜(받은 만큼) 기준"**이 된다.
- **내 데이터는 내 앱에만 있음 → 복제 불가**
- **쓸수록 나만을 위해 정확 → 스위칭 비용 → 재사용 고정**
- 3전환: ① 평균 → **호혜** ② 1회 계산 → **관계 원장** ③ 금액 → **완주(봉투+멘트)**

> ※ 장부앱(수수·주고받고)이 "기록"은 점유. 우리 차별 = 그들은 **기록만**, 우리는 원장을 **"다음에 얼마 낼지 판단 + 봉투문구 완주"에 자동 활용** + **청첩장 붙여넣기로 자동 기록**. "메모장"이 아니라 "기억해서 판단하는 비서".

### 심사 "그냥 계산기 아니냐" 30초 방어
> "계산기는 하객수 × 평균을 하죠. 저흰 다릅니다. 계산기는 **당신이 예전에 그 사람에게 얼마 냈는지 모릅니다.** 경조사의 진짜 기준은 평균이 아니라 **호혜 — 받은 만큼 낸다**예요. 그리고 계산기는 금액에서 끝나지만, 실제로 막히는 건 그 다음 — 봉투에 뭐라 쓸지, 종교 다른데 뭐라 인사할지. 저흰 청첩장 한 장에서 금액·봉투·멘트까지 끝냅니다. 계산기는 도구고, 저흰 **당신의 경조사 관계를 기억하는 비서**입니다."

## 3. 킬러 기능

### 🔑 "청첩장을 계산하지 말고 붙여넣으세요"
경쟁(계산기 앱·검색)은 전부 사용자가 숫자·관계·지역을 **입력**해야 함 = "계산기". 우리는 카톡으로 받은 초대장을 **그대로 복붙** → 식장·날짜 자동 파싱 → tool 도미노 → 결과 카드 1장. **"입력"이 아니라 "붙여넣기".**

- 파싱은 MCP tool이 아니라 **챗봇 자연어 레이어**에서(LLM이 텍스트→구조화). MCP tool은 확정값 받아 **결정적 계산만** = 안정성 확보.
- 캡처 이미지도 OCR → 동일 파이프라인. 파싱 실패 시 "식장 이름만 알려주세요" 폴백(절대 안 멈춤).
- 결혼 = 밝게/이모지, 장례 = 절제/예법 **톤 자동 전환**(계산기엔 없는 정서 지능).

## 7. 공유카드 3종 (본선투표30 직결 — 놀맵 교훈 강제)

| 카드 | 디자인 | 바이럴 훅 |
|---|---|---|
| ① 봉투 컨닝페이퍼(메인) | 세로4:5, 한지+인주, 큰 숫자+봉투문구 세로쓰기 | "식장 화장실서 몰래 보는 컨닝페이퍼" = 실용 효능감 |
| ② "이 금액인 이유" 근거카드 | 영수증 스타일, 계산식 노출 | "이래서 10만이래" 단톡 공유 |
| ③ 시즌 브리핑 | 캘린더 그리드+총합 | "나 이번 가을 30만 나감ㅠ" 경조사 거지썰 = 공감 자발공유 |

**놀맵 5001 교훈 강제**: 카톡 공유 imageUrl은 **정적 프리렌더 PNG만**(동적 쿼리스트링 금지), 정적 fallback 이미지 항상 세팅.
**본선 전략**: 심사장에서 "제 청첩장 하나 붙여넣어볼게요" → 3초 만에 카드 뿅 라이브 시연(관객이 자기 폰 따라하게).

## 9. 경쟁 우위 (품질 기준선 + 차별화 TOP5)

### 넘어야 할 품질 기준선
수수 수준 완성도 · 토스/카카오페이 "3초 컷" 심플(질문 3개 이내) · 카카오페이 봉투 수준 비주얼 · 숫자마다 출처 표기.

### 차별화 TOP5 ("이 서비스는 이걸 못한다")
1. 사전 의사결정 원스톱 — **장부앱은 기록만**
2. 관계 호혜 기준 — **챗봇·계산기는 당신이 예전에 얼마 냈는지 모름**
3. 봉투문구+멘트 대필 — **핀테크는 송금만, 챗봇은 규범 틀림**
4. 축의금 판단 공백 정면 — **카카오페이도 만우절 개그로만**
5. MCP 대화형 — **앱 설치·폼 입력 마찰 0**

## 10-B. 해외 벤치마크 — 이중 무주공산 (2026-07 리서치)

### 메신저 안 MCP는 카카오가 세계 선구자
- 해외 4대 메신저(WhatsApp 30억·Telegram 10억·LINE 9400만·WeChat 14억)의 MCP는 **거의 전부 "개발자가 외부에서 자기 계정 자동화하는 백엔드 브리지"** — 일반 소비자가 톡 안에서 MCP 앱을 쓰는 공식 플랫폼 아님.
- **WhatsApp은 오히려 2026.1 Meta가 범용 AI 금지**(ChatGPT·Perplexity 축출). Slack만 1st-party MCP(2026.2)로 근접하나 엔터프라이즈용.
- **카카오 PlayMCP = "소비자 메신저 안 서드파티 MCP 대화형 마켓"의 세계 유일/선구자.** PlayTools(소비자 마켓) 2026 상반기 = 본선 시점 가동. → 우리는 이 선점 인프라 위에 올라탐.

### 경조사 통합 비서 = 서구에도 없음
- 서구는 관계추천(DreamGift·Escargot)·리마인더(Mesh/Dex 개인CRM)·송금(Venmo·Hallmark+Venmo)·레지스트리(Zola)가 **전부 파편화**. 하나로 통합된 대화형 경조사 비서 **전무**. 가장 근접한 MCP = B2B 기프팅(Goody), 개인 경조사 아님.
- **장례(부의금)는 서구 대응물 특히 없음**(memorial donation으로 대체). "관계→금액" 로직은 서구가 약함(그들은 "취향→상품").

### 승자 공식 = 큐레이션은 되고 결제는 미성숙
- ChatGPT Apps도 **결제는 2026.3 후퇴**, 여행 실예약 8%. → AI는 "얼마·뭐라 쓸지 판단"에 집중, **송금은 카카오페이/선물하기에 위임**하는 하이브리드가 정답 = 우리 설계(딥링크까지만)와 일치.
- **한 줄: 수요·문화는 한국이 세계 최적, 메신저+MCP는 세계 초기 = 얼리무버 프리미엄.** 신규성보다 "문화특화+통합"으로 포지셔닝.

### 추가 확정 정보 (본선·수익화에 직결)
- **PlayMCP 실사용 순위(공식)**: 나와의채팅방 > 카카오맵 > 학교급식 > 날씨 > 멜론 > 미국주식. **소비자 일상 서비스가 실제로 대화형 MCP로 돎** = 우리 카테고리(생활밀착) 정합.
- **The Knot이 "축의금 계산기"를 실제 운영** = 이 페인은 문화특수 아니라 **보편적**(시장 존재 방증). ⚠️ **단 그건 레지스트리 유입 미끼이지 수익모델 아님 → 경조사 "계산 단독" BM은 서구에서도 약함.** 우리도 관계기억(해자)에 더해 **카카오 선물하기·페이 송금 연결(거래 훅)**이 완결성·수익화 관건.
- **관계 상호성(호혜) 정량화는 한국이 서구보다 훨씬 첨예** = 우리 "관계기억 원장" 해자의 문화적 근거. (서구 personal CRM Monica/Dex도 선물기록은 하나 "경조사 적정액 계산"과 결합 안 함)
