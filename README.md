# 얼마낼까 — 경조사 익명 커뮤니티

> 카카오 AGENTIC PLAYER 10 MCP 공모전 출품작
> **[얼마낼까 - 경조사 익명 커뮤니티]** · "당신만 그런 거 아니에요."

경조사(결혼·장례·돌잔치) 축의금·조의금을 도와주는 카카오톡 MCP. 핵심은 **"남들은 얼마 냈나"** — 실제 사람들이 익명으로 등록한 금액을 상황별로 집계해 보여주고, 같은 고민을 한 사람들의 익명 경험을 함께 전한다.

## 왜?
- 경조사 페인의 본질은 "얼마"(숫자)가 아니라 **"나만 이렇게 눈치보나"**(불안·비교).
- 그래서 ①남들은 얼마 냈나(익명 통계) + ②같은 고민 익명 경험(위안)을 한 번에 준다.
- 여기에 ③내 상황 적정액 추천, ④봉투 문구·멘트, ⑤관계 기억(호혜)까지 얹어 **경조사 결정을 끝까지** 도와준다.

## 데이터·안전
- **익명 집계 전용**: 개별 기록 원문 노출 안 함(중앙값·최빈값·분포만). 식장명·정확날짜·실명 수집 안 함.
- **k-익명성**: 표본 5건 미만은 통계 미산출("표본 부족" 정직 표기).
- **조작 방어**: 중앙값·최빈값(평균 대신), IQR 이상치 트림, 금액 범위 강제(1만~100만), 신뢰도 배지.
- **시드**: 공개 통계(신한 1만명·카카오페이 실측·인크루트 844명)로 콜드스타트 해결. 출처 있는 것만.

## 실행
```bash
npm install
npm run build
npm start            # stdio (로컬)
MODE=http npm start  # http (PlayMCP 배포용, /mcp 엔드포인트)
```

## 배포 (PlayMCP in KC · Git 소스 빌드)
GitHub에 올린 뒤 PlayMCP in KC에 저장소 URL 등록 → 카카오가 Dockerfile로 빌드 → Endpoint 발급.
런타임 키 불필요(통계는 내장 시드). ⚠️ 크라우드 데이터는 컨테이너 로컬에 쌓임 — 실서비스 영속화는 `src/storage.ts`를 외부 DB로 교체.
