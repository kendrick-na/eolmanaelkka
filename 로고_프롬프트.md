# 「얼마낼까」 앱 아이콘 — 제미나이(나노 바나나) 프롬프트 (브랜딩 재설계본)

> **핵심 통찰: 봉투는 소품, 주인공은 사람. "혼자"를 "여럿"으로 바꾸는 게 이 서비스의 전부.**
> 지금까지 실패 = "봉투=축의금" 1:1 직역(상품을 그림). 우리 심장 = "봉투 앞 3초 망설임" + "혼자가 아닌 안도".
>
> **스타일(세금맛집 앱아이콘 계승):** 카카오 옐로우(#FEE500) 배경 + 짙은 브라운(#3C1E1E) 단색 실루엣 + 한 장면(scene) + 텍스트 0.
> **3원칙:** ①사람(손)을 넣는다 ②복수성(여럿)을 시각화 ③한 장면으로 어우러짐.
> **배제:** NO food, NO credit card, NO coins, NO circular badge, NO ribbon, NO text/letters/numbers.
>
> **📐 규격: 600x600 이상, png/jpg/jpeg/gif.** → 1024로 뽑고 600 리사이즈본 함께 저장.
> **제작 팁:** AI Studio 에러 시 모델을 "Gemini 2.5 Flash Image"(무료)로. 또는 gemini.google.com 앱에서. 앱스토어 축소(120px)에서 뭉침 확인 필수.

## 추천: **A(메인) + C(감정) + D(안전) 3안 뽑아 비교.** 셋 다 "봉투≠주인공, 사람·복수성이 주인공"을 지킴.
- A = 서비스 정체성 완성도 최고(3손=남들은 얼마+통계+고민). 단 축소 시 뭉침 주의(손 단순화·물음표 크게).
- C = 감정 밀도 최강, 축소 내구성 좋음. 단 "커뮤니티/비교"는 덜 읽힘.
- D = 가장 안전, 어떤 크기도 안 뭉침. 온기는 약간 덜함.
