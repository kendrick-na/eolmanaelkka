# 1단계 — PlayMCP in KC 서버 배포 (Git 소스 빌드)

세금맛집과 완전히 동일. 카카오가 GitHub 저장소의 Dockerfile을 서버에서 직접 빌드해줌 (내 맥에 Docker 불필요).

1. **https://playmcp.kakaocloud.io** 접속 → 카카오 계정 로그인
2. **"+ 새 MCP 서버 등록" → "Git 소스 빌드"** 선택
3. 입력:

| 항목 | 입력값 |
|---|---|
| **MCP 서버 이름** | `eolmanaelkka` (소문자·숫자·하이픈만) |
| **설명** | `남들은 얼마 냈을까? 경조사 축의금·조의금을 익명으로 확인하는 커뮤니티 MCP` |
| **Git URL** | `https://github.com/kendrick-na/eolmanaelkka.git` |
| **브랜치 / ref** | `main` |
| **Dockerfile 경로** | `Dockerfile` |
| **PAT** | 비움 (Public 저장소) |

4. **"등록하기"** → Status: Starting → **Active** 되면 성공
5. Active된 서버 클릭 → **Endpoint URL 복사** (예: `https://eolmanaelkka.playmcp-endpoint.kakaocloud.io/mcp`)
   - → 이 URL을 2단계에서 씀. **Claude에게 알려주면 검증해줌**

eolmanaelkka
```

### 대표 이미지

```
축의금 5만원? 10만원? 봉투 앞에서 혼자 고민하지 마세요. 얼마낼까는 실제로 사람들이 낸 금액을 익명으로 모아 "남들은 이 상황에 얼마 냈는지"를 통계로 보여주는 경조사 커뮤니티입니다.

관계·식장·나이에 맞는 적정 축의금/조의금 추천, 봉투에 뭐라 쓸지(종교별 봉투 문구·조문/축하 멘트), 이 경조사 갈까 말까 판단까지. 예전에 받은 것을 기억해 "받은 만큼" 챙기도록 도와드립니다.
